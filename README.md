## Instalasi
Requirement : 
`PHP >8.2`
`npm`
`mysql`
`phpmyadmin : (optional)`
[Composer 2.8](https://getcomposer.org/download/2.8.2/composer.phar)

### Menjalankan Program
Install composer didalam project anda
```shell
composer install
```
jalankan perintah migrasi
```shell
php artisan migrate
```
Siapkan `.env` yang sudah tersedia pada `.env.example`
Lalu jalankan dengan menggunakan 
```shell
php artisan serve
```
```shell
npm run dev
```
 > **WARNING**
 > Pastikan mysql telah berjalan pada localhost !
